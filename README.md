## ダーツの点数、勝敗を管理するためのアプリ
- プレイするゲームモードを選択
- 表示されるユーザーが投げたダーツの刺さった場所をクリック
- アプリが自動で次に投げる人、投げる本数、点数、勝敗を管理します。

## 使用言語（フレームワーク）
- フロントエンド：JavaScript
- バックエンド：Python(Flask)

## 目的
- 自宅に設置したダーツボードでプレイする際、点数管理が難しかったため作成
- APIの作成
- Flaskの習得
